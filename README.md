# QA - Relatório

Este repositório tem como objetivo demonstrar conceitos essenciais estudados e também utilizados por profissionais de QA (Quality Assurance) durante sua trajetória no desenvolvimento de software.

## Conteúdos abordados

- Tipos de testes

- Ciclo de vida do software

- Ciclo de vida de bugs

- Pirâmide de testes

- Exemplos de bugs

- Glossário de termos de QA

## Objetivo do projeto

Este projeto foi criado para a consolidação de conhecimentos iniciais em QA, servindo como base para projetos futuros, tanto iniciais quanto mais avançados, envolvendo testes, automação e testes de API.
